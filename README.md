NGO Fund Tracker
The NGO Fund Tracker is a web-based application developed primarily with HTML (≈80%) for the frontend structure and styling, combined with PHP for backend logic and data handling. The project is designed to run locally on the XAMPP server, making it simple to set up and test in a development environment.
✨ Features
• 	Frontend: Built mostly with HTML for clear layouts and user-friendly navigation.
• 	Backend: PHP scripts handle server-side operations and database connectivity.
• 	Local Hosting: Runs seamlessly on XAMPP, accessible via .
• 	Database Support: MySQL integration through phpMyAdmin for storing and managing NGO fund data.
